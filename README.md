# OFFCHAT – Peer-to-Peer Messaging Without Internet

OFFCHAT is an offline messaging application that enables communication without internet or mobile networks using Bluetooth Low Energy (BLE) Mesh technology. The system supports secure peer-to-peer messaging, multi-hop communication, group chats, and offline attendance management.

## Features

* Offline messaging using BLE Mesh
* Peer-to-peer communication
* Multi-hop message relaying
* Private and global chat
* Group chat support
* Offline attendance system
* AES-128 encrypted messaging
* Low power consumption

## Technologies Used

* Kotlin
* Android Studio
* Bluetooth Low Energy (BLE)
* BLE Mesh Networking
* AES-128 Encryption
* DataStore / SharedPreferences

## System Architecture

The application follows a layered architecture including:

* User Layer
* Presentation Layer
* Application Layer
* Security Layer
* Local Storage
* Bluetooth Mesh Network Layer

## Use Cases

* College and campus communication
* Disaster and emergency response
* Remote construction sites
* Large outdoor events
* Areas without internet connectivity
